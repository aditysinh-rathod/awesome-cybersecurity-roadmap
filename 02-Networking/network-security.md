# 🛡️ Network Security

## Overview

Network Security protects systems, devices, and data from unauthorized access and attacks.

---

# Security Controls

- Firewalls
- IDS
- IPS
- VPN
- VLAN
- MFA
- NAC
- Proxy Servers

---

# Common Attacks

| Attack | Description |
|---------|-------------|
| DoS | Denial of Service |
| DDoS | Distributed DoS |
| ARP Spoofing | Fake ARP Messages |
| DNS Spoofing | Fake DNS Replies |
| MITM | Intercept Communication |
| Packet Sniffing | Capture Packets |

---

# Best Practices

- Strong Passwords
- Network Segmentation
- Logging
- Encryption
- Updates
- Least Privilege

---

# Key Takeaways

Security is layered.

Protect endpoints, users, applications and networks together.
