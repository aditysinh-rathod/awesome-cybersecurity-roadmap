# 📖 Overview

Wireshark is the world's most popular **network protocol analyzer**.

It allows you to capture, inspect, and analyze packets travelling across a network.

---

# Features

- Live Packet Capture
- Deep Packet Inspection
- Protocol Analysis
- Network Troubleshooting
- Malware Investigation
- Incident Response
