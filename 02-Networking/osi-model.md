# 📖 Overview

The **Open Systems Interconnection (OSI) Model** is a conceptual framework that explains how data moves between devices over a network.

It consists of **7 layers**, each responsible for a specific function.

---

# 📊 OSI Layers

| Layer | Name | Example |
|--------|------|----------|
| 7 | Application | HTTP, FTP, DNS |
| 6 | Presentation | SSL/TLS, Encryption |
| 5 | Session | NetBIOS |
| 4 | Transport | TCP, UDP |
| 3 | Network | IP |
| 2 | Data Link | Ethernet, MAC |
| 1 | Physical | Cable, Fiber, Wi-Fi |

# 📚 Key Takeaways

- OSI has 7 layers.
- Each layer performs a specific function.
- Cybersecurity professionals use the OSI model to troubleshoot and analyze attacks.
