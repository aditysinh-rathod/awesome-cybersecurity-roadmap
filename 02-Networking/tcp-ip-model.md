# 📖 Overview

The **TCP/IP Model** is the practical networking model used on the Internet.

It contains **4 layers**.

---

# 📊 Layers

| Layer | Protocols |
|--------|-----------|
| Application | HTTP, FTP, DNS |
| Transport | TCP, UDP |
| Internet | IP, ICMP |
| Network Access | Ethernet, Wi-Fi |

---

# Comparison

| OSI | TCP/IP |
|-----|--------|
| Application | Application |
| Presentation | Application |
| Session | Application |
| Transport | Transport |
| Network | Internet |
| Data Link | Network Access |
| Physical | Network Access |

---

# TCP vs UDP

| TCP | UDP |
|-----|-----|
| Reliable | Fast |
| Connection Oriented | Connectionless |
| Error Checking | Minimal Checking |
| Ordered | Unordered |

Examples

TCP

- HTTP
- HTTPS
- SSH

UDP

- DNS
- VoIP
- Streaming

# Key Takeaways

- TCP is reliable.
- UDP is faster.
- Internet communication relies on the TCP/IP model.
