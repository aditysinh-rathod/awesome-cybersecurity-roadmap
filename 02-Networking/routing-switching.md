# 📖 Overview

Routers and switches are the backbone of modern computer networks.

- **Switch** → Connects devices within the same LAN.
- **Router** → Connects different networks together.

---

# Network Devices

| Device | Function |
|---------|----------|
| Hub | Broadcasts to all devices |
| Switch | Uses MAC Addresses |
| Router | Uses IP Addresses |
| Firewall | Filters Traffic |
| Access Point | Wireless Connectivity |

---

# Router vs Switch

| Router | Switch |
|---------|---------|
| Layer 3 | Layer 2 |
| Uses IP | Uses MAC |
| Connects Networks | Connects Devices |
| Internet Gateway | LAN Communication |

---
