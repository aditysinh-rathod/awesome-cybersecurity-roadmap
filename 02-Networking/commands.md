# 💻 Essential Networking Commands

## Linux

| Command | Description |
|---------|-------------|
| ip addr | Show IP Address |
| ip route | Show Routing Table |
| ping | Connectivity Test |
| traceroute | Route Path |
| ss | Active Connections |
| netstat | Network Statistics |
| arp | ARP Table |
| hostname | System Name |
| curl | HTTP Requests |
| wget | Download Files |
