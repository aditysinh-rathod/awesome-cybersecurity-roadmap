# 📖 Overview

Networking is the backbone of cybersecurity. Understanding how devices communicate, how packets travel, and how protocols work is essential for both defensive and offensive security.

This module covers networking fundamentals through structured notes, practical labs, diagrams, command references, and security concepts.

---

# 🎯 Learning Objectives

- Understand the OSI & TCP/IP models
- Learn IP addressing and subnetting
- Understand common ports and protocols
- Explore DNS, HTTP, HTTPS
- Learn routing and switching
- Analyze packets with Wireshark
- Secure networks with firewalls
- Build a strong networking foundation for cybersecurity
