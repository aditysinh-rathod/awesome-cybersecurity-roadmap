# 📖 Overview

Linux powers the majority of servers, cloud platforms, penetration testing distributions, and security infrastructure.

This module documents my Linux learning journey with cybersecurity in mind through structured notes, command references, practical labs, automation scripts, and hands-on exercises.

---

# 🎯 Learning Objectives

- Understand Linux fundamentals
- Navigate the Linux file system
- Manage users and permissions
- Work with processes and services
- Secure SSH access
- Learn Bash scripting
- Analyze logs
- Automate repetitive tasks
