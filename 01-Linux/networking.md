# 🌐 Linux Networking

## 📖 Overview

Networking is one of the most important Linux skills for cybersecurity.

---

# Essential Commands

| Command | Purpose |
|----------|---------|
| ip a | Show IP |
| ping | Test connectivity |
| traceroute | Trace route |
| ss | Network sockets |
| netstat | Connections |
| hostname | System hostname |
| curl | HTTP requests |
| wget | Download files |
