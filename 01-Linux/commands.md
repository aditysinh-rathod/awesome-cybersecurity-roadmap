# 💻 Essential Linux Commands

## Navigation

| Command | Description |
|---------|-------------|
| `pwd` | Print current directory |
| `ls` | List files |
| `ls -la` | List hidden files |
| `cd` | Change directory |

---

## File Management

| Command | Description |
|---------|-------------|
| `mkdir` | Create directory |
| `touch` | Create file |
| `cp` | Copy files |
| `mv` | Move/Rename |
| `rm` | Remove |
| `find` | Search files |

---

## Permissions

| Command | Description |
|---------|-------------|
| `chmod` | Change permissions |
| `chown` | Change ownership |
| `umask` | Default permissions |
