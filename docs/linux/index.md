## 📖 Module Overview

Linux is the most important operating system in cybersecurity. Nearly every security tool, server, cloud platform, and penetration testing environment runs on Linux.

In this module you'll learn:

- Linux Fundamentals
- Terminal Commands
- File System
- Users & Groups
- Permissions
- Bash Scripting
- SSH
- Process Management
- Package Management
