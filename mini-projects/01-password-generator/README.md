## 📖 Overview

A secure password generator built with Python.

It creates random passwords using uppercase letters, lowercase letters, numbers, and special characters.

---

## ✨ Features

- Random password generation
- Custom password length
- Uppercase support
- Lowercase support
- Numbers
- Symbols
- Input validation
- Command-line interface

---

## 🚀 Usage

```bash
python main.py
```

Example

```text
Password Length: 16

Generated Password:

P@7#Lm9!Qz2$RwX8
```
